The ``MyList`` class inherits from ``list`` and adds a method
to print a sorted version of the list without modifying the original.

>>> MyList = __import__('1-my_list').MyList
>>> my_list = MyList()
>>> isinstance(my_list, list)
True
>>> my_list.append(1)
>>> my_list.append(4)
>>> my_list.append(2)
>>> my_list.append(3)
>>> my_list.append(5)
>>> print(my_list)
[1, 4, 2, 3, 5]
>>> my_list.print_sorted()
[1, 2, 3, 4, 5]
>>> print(my_list)
[1, 4, 2, 3, 5]

Negative numbers still work:

>>> my_list = MyList()
>>> my_list.append(-1)
>>> my_list.append(3)
>>> my_list.append(0)
>>> print(my_list)
[-1, 3, 0]
>>> my_list.print_sorted()
[-1, 0, 3]
>>> print(my_list)
[-1, 3, 0]

Empty list:

>>> my_list = MyList()
>>> print(my_list)
[]
>>> my_list.print_sorted()
[]
>>> print(my_list)
[]
