# Python ABC - Abstract Animal

## Task 0: Abstract Animal Class and its Subclasses

### Objective
- Create an abstract base class `Animal` using Python's `abc` module.
- Add an abstract method `sound()`.
- Implement subclasses:
  - `Dog.sound()` returns `"Bark"`
  - `Cat.sound()` returns `"Meow"`

### Files
- `task_00_abc.py`: Implementation of `Animal`, `Dog`, `Cat`
- `main_00_abc.py`: Simple test (optional)
